# expo-smart-image

React Native Image Cache, Progressive Loading, Zoom, Loading Indicator based on Expo

## Installation

```sh
npm install expo-smart-image
```

## Usage

```js
import { multiply } from "expo-smart-image";

// ...

const result = await multiply(3, 7);
```
